## 🌟 Featured Projects

### Production RAG System
Built a Retrieval-Augmented Generation system using **LangChain**, **ChromaDB**, and the **OpenAI API** for grounded question answering over a document corpus.

**Highlights**
- Hybrid retrieval with **semantic search + BM25**
- Added a **reranker** to improve answer precision
- Evaluated quality with **RAGAS**
- Deployed with **Streamlit** on **Hugging Face Spaces**

**Tech:** `Python` `LangChain` `ChromaDB` `OpenAI API` `RAGAS` `Streamlit`

### Football Analysis System
Built a computer vision pipeline for player, referee, and ball tracking from match footage.

**Highlights**
- Object detection with **YOLO** and **OpenCV**
- Team assignment using **K-Means Clustering**
- Optical flow for camera motion compensation
- Perspective transformation for real-world metrics like **distance** and **speed**

**Tech:** `Python` `YOLO` `OpenCV` `NumPy` `scikit-learn` `Docker` `Pytest`

### Hospital Readmission Predictor
Created an end-to-end machine learning pipeline for predicting diabetic patient readmission.

**Highlights**
- Feature engineering and data balancing with **SMOTE** and **ADASYN**
- Compared and optimized 8 classification models
- Used **Monte Carlo Cross-Validation** for stability and robustness

**Tech:** `Python` `scikit-learn` `XGBoost` `Pandas` `NumPy`
